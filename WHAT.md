### Project 2: Automated Data Quality Auditor

In this project, the main goal is to feed it a dataset, check for data quality issues, and spit a report.

It aims to check:
    - null values
    - duplicate rows
    - mismatched dates
    - outliers

It will be expected to have an output of a clean summary result in an HTML or a PDF.
